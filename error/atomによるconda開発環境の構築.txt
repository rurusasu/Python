*minicondaをインストール*
コマンドプロンプトにて以下のコマンドを実行
(python3.6以外の場合は数字をそのバージョンに変更)
conda create -n py36_dw python=3.6
conda info -e

*以下のようなコメントが出れば成功*

# conda environments:
#
py36_dw                  C:\Miniconda3\envs\py36_dw
root                  *  C:\Miniconda3

*仮想環境の切り替え*
切り替え後以下のコマンドを実行
conda install jupyter
以下のコマンドを実行してjupyterカーネルを取得
ipython kernel install --user --name py36_dw --display-name Python3-DataWrangling
以下のコマンドを実行してカーネルを確認
jupyter kernelspec list

*以下のようなコメントが出れば成功*
[USERNAME]にはユーザ名が入る.

jupyter kernelspec list
Available kernels:
  py35_dw    C:\Users\[USERNAME]\AppData\Roaming\jupyter\kernels\py35_dw
  python3    C:\Miniconda3\envs\py36_dw\share\jupyter\kernels\python3

*conda installで以下のパッケージを入手*
conda install numpy
conda install scipy
conda install matplotlib

*JupyterNoteBookの設定*
コマンドプロンプトを起動し以下のコマンドを実行
jupyter notebook --generate-config

C:\Users\User\.jupyter内にjupyter_notebook_config.pyファイルが作成される.

*jupyter_notebook_config.pyをメモ帳で開く*
214行付近
#c.NotebookApp.notebook_dir = ''
を以下のように変更
\c.NotebookApp.notebook_dir = 'D:\自宅用\Python\Jupyter_Notebook'

*他にも以下のようなことが紹介されていた*
220行付近
220 #c.NotebookApp.open_browser = True
を以下のように変更
c.NotebookApp.open_browser = False
これにより勝手にブラウザが立ち上がるのを防ぐ.

*220行を操作する場合以下の操作も平行して行うこと!!!*
JupyterNoteBookショートカットを右クリック→その他→ファイルの場所を開く
→JupyterNotのプロパティを表示→リンク先の「%USERPROFILE%」を削除→適用→続行→OK

*jupyterの設定は以上*

*atomをインストール*
Atom内で以下のパッケージをインストール
atom-beautify
autocomplete-paths
autocomplete-python
highlight-cursor-line
hilight-selected
japanese-menu
linter-flake8
script
hydrogen

*Atom内でプログラムを作成*
プログラム作成終了後　CTRL+SHIFT+ALT+ENTERを押す.
プログラムが走れば成功.
